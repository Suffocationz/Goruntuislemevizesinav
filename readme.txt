  Parmak Hareketleriyle Ekran Parlaklığı Kontrolü

  Proje Tanımı

  Bu projede, başparmak ve işaret parmağı arasındaki mesafe kullanılarak bilgisayar ekran parlaklığı kontrol edilmiştir. Kamera aracılığıyla el hareketleri algılanır, iki parmak arası mesafe hesaplanır ve bu mesafeye bağlı olarak ekran parlaklığı artırılır veya azaltılır.

Kullanılan Kütüphaneler

- Python 3.11
- OpenCV 
- MediaPipe 
- NumPy
- Screen Brightness Control
